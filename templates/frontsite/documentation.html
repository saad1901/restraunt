Getting Started with TastyBits

Welcome to TastyBits, your all-in-one restaurant management solution.

TastyBits helps restaurant owners and managers streamline their operations, from order management to inventory tracking and staff scheduling. This documentation will guide you through setting up and using all features of our platform.

System Requirements

  • Modern web browser (Chrome, Firefox, Safari, Edge)
  • Internet connection
  • For optimal mobile experience: Android 8.0+ or iOS 12+

Account Access

To access your TastyBits account:

  1. Visit TastyBits Login
  2. Enter your username and password
  3. Click "Sign In"

If you've forgotten your password, click the "Forgot Password" link on the login page.

New to TastyBits? Sign up for a free trial to explore all features.

Setup Guide

Follow these steps to set up your TastyBits account for the first time:

Restaurant Profile

Begin by setting up your restaurant profile with basic information:

  • Navigate to Settings > Restaurant Profile
  • Add your restaurant name, address, contact information, and logo
  • Set your business hours and special closing days
  • Configure tax settings applicable to your location
Menu Setup

Create your menu categories and items:

  • Go to Menu > Categories to create menu sections (Appetizers, Main Course, etc.)
  • Add items to each category with prices, descriptions, and images
  • Create variations and modifiers for customizable items
  • Set item availability and special offers
Table Configuration

Set up your floor plan and tables:

  • Access Tables > Floor Plan
  • Create different sections or areas in your restaurant
  • Add tables with appropriate numbers and seating capacity
  • Arrange tables according to your actual floor layout
Staff Accounts

Add your staff members and set their access levels:

  • Navigate to Staff > Manage Staff
  • Add staff members with their contact information
  • Assign appropriate roles (Manager, Waiter, Cashier, etc.)
  • Set permissions for each role to control access to different features

Menu Management

The Menu Management section allows you to create, organize, and update your restaurant's offerings.

Creating Categories

Categories help organize your menu items into logical sections:

  1. Go to Menu > Categories
  2. Click "Add Category"
  3. Enter a name and optional description
  4. Set the display order to control how categories appear on the menu
  5. Click "Save"

Adding Menu Items

To add items to your menu:

  1. Navigate to Menu > Items
  2. Click "Add Item"
  3. Fill in the item details:
    • Name and description
    • Category
    • Price
    • Preparation time
    • Upload an image (recommended size: 500Ã—500 pixels)
  4. Add variations if needed (e.g., Small, Medium, Large)
  5. Add modifiers or extras (e.g., toppings, sides)
  6. Click "Save Item"
Tip: Regular menu updates keep customers engaged. Consider seasonal items and specials to increase interest.

Troubleshooting

Solutions for common issues you might encounter while using TastyBits.

If you're having trouble logging in:

  1. Check that you're using the correct username and password
  2. Ensure Caps Lock is not enabled
  3. Try resetting your password using the "Forgot Password" link
  4. Clear your browser cache and cookies
  5. Try using a different browser

If you still can't log in, please contact our support team.

If orders are not reaching the kitchen display:

  1. Check your internet connection
  2. Verify that the kitchen display system is online
  3. Ensure the staff member has the correct permissions to send orders
  4. Check if the order status is "Sent" or still "Draft"
  5. Try refreshing the kitchen display page

For persistent issues, restart the kitchen display device or contact support.

If your reports show unexpected or incorrect data:

  1. Check the date range selected for the report
  2. Verify that all transactions were properly closed/settled
  3. Look for any voided transactions that might affect totals
  4. Ensure all staff members are using the system correctly
  5. Try clearing the cache or using a different browser

If issues persist, please contact our support team with examples of the incorrect data.
